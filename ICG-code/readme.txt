Source code for LightSlice:
===========================================

Content
-------------------------------------------
Source package include:
1. Implementation of LightSlice
2. Implementation of Lightcut
3. Implementation of Multidimentional Lightcut
4. Implementation Matrix Row Column Sampling

Folder
--------------------------------------------
common
	Basic data structure

far
	Scene management and ray engine
	
mcs
	manylight projects
	

Compiling Enviroment
-------------------------------------------
Windows Operation System
Visual Studio 2010
CMake 2.6
tbb


Setup
--------------------------------------------
1. Install CMake 2.6
2. Run mcs/CMake64.bat
3. Open mcs/obj/msvc/manylight.sln




Projects
---------------------------------------------
lightcut:		Lightcut
mlightcut:		Multidimentional Lightcut
mrcs:			Matrix Row Column Sampling
ccmat:			LightSlice
